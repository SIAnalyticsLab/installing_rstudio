üß≠ Installing and Using RStudio

A simple step-by-step guide for beginners using R and Quarto

Step 1: Download and Install R

Before installing RStudio, you must first install R, the programming language itself.

  1. Go to the official R website: https://cran.r-project.org
  2. Click Download R for your operating system:
    • Windows ‚Üí ‚ÄúDownload R for Windows‚Äù
    • macOS ‚Üí ‚ÄúDownload R for macOS‚Äù
    • Linux ‚Üí ‚ÄúDownload R for Linux‚Äù
  3. Choose the latest version and follow the installation wizard.
  4. Once installed, confirm R is working by typing:
version

Step 2: Download and Install RStudio

  1. Visit the RStudio (Posit) website: https://posit.co/download/rstudio
  2. Scroll down and select the RStudio Desktop (Free) version.
  3. Choose your operating system and download the installer.
  4. Run the installer and follow the setup wizard with default settings.

Step 3: Launch RStudio

Open RStudio from your applications or Start menu. You‚Äôll see four main panes:

  • Source Editor (top-left) ‚Äî where you write and edit code
  • Console (bottom-left) ‚Äî where R executes your commands
  • Environment/History (top-right) ‚Äî tracks data and variables
  • Files/Plots/Packages/Help/Viewer (bottom-right) ‚Äî for navigation and visualizations

Step 4: Write and Run Your First R Script

  1. Click File ‚Üí New File ‚Üí R Script.
  2. Type this simple command:
print("Hello, RStudio!")

Press Ctrl + Enter (Windows) or Cmd + Enter (Mac) to run the line. The output will appear in the Console pane.

Step 5: Install Essential R Packages

Enhance your workflow by installing popular packages for data science and visualization:

install.packages(c("tidyverse", "ggplot2", "plotly", "dplyr", "quarto"))

Load a package in your session using:

library(ggplot2)

Step 6: Connect and Use Quarto in RStudio

Quarto is built into RStudio (2022.07+). To confirm:

  1. Click File ‚Üí New File ‚Üí Quarto Document.
  2. Select HTML as the output format and click Create.
  3. Click Render to generate your first Quarto report.

If you don‚Äôt see the ‚ÄúQuarto‚Äù option, install it manually:

install.packages("quarto")

Then restart RStudio.

Step 7: Organize Your R Projects

  1. Create a project folder (e.g., MyQuartoDashboard).
  2. In RStudio, go to File ‚Üí New Project ‚Üí Existing Directory.
  3. Select your folder and click Create Project.
  4. Keep all files ‚Äî .R, .qmd, .csv ‚Äî in one place for a reproducible workflow.

Step 8: Save and Export Your Work

  • Save scripts with Ctrl + S or Cmd + S.
  • Export plots from the Plots tab ‚Üí Export ‚Üí Save as Image or PDF.
  • Render .qmd documents as HTML, PDF, or Word using the Render button.

  • All tools needed to publish professional, reproducible analytics
